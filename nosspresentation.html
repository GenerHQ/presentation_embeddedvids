Creating a New Ticket

Demo

Step-by-Step

  1. Click "NEW TICKET" button
  2. Enter Service Request No. (6 digits exactly)
  3. Enter Requested By (letters and spaces only)
  4. Select Branch/Department from auto-suggest datalist
  5. Select Assigned Support (multiple checkboxes)
  6. Select Problem Type (multiple checkboxes)
  7. Set Ticket Status & Current Action (auto-linked)
  8. Enter Time Started / Time Completed (auto-calculates Time Difference)
  9. Enter No. of Endpoints (auto-calculates Time Per Endpoint)
  10. Click SAVE TICKET
Auto-Calculation: Time Difference (hh:mm:ss) and Time Per Endpoint are calculated automatically!

Updating & Deleting Tickets

Demo

Update Ticket

  1. Click the Edit (pencil) icon in Actions column
  2. Modify any field in the Update Modal
  3. System automatically preserves modification history in History Logs
  4. Click UPDATE RECORD to save changes

Delete Ticket

  1. Click the Trash icon in Actions column
  2. Enter optional deletion reason in the prompt
  3. Ticket moves to Archived Tickets (soft delete)
  4. ⚠️ Ticket is NOT permanently deleted - can be restored later!

Archived Tickets (Restore)

Demo

How to Restore a Deleted Ticket

  1. Click "Archived Tickets" in the navigation bar
  2. Click REFRESH PAGE to load deleted records
  3. Search by ITSM number if needed
  4. Click the Restore (undo) icon in the Actions column
  5. Confirm restoration in the modal dialog
  6. Ticket appears back in active tickets with restoration record in history!
Warning: Permanent delete removes forever - use with caution! Regular delete is safe (soft delete).
